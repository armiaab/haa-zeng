This code was purposed to reproduce [Quantum Neural Network Inspired Hardware Adaptable Ansatz for Efficient Quantum Simulation of Chemical Systems](https://pubs.acs.org/doi/10.1021/acs.jctc.3c00527) by Zeng et. all with some modification using qiskit instead Q^2 chemistry. This code only works on POSIX system since pyscf doesn't support Windows

## Getting Started

python3 -m pip install -r requirements.txt
